D (Debug)    - Debug information (e.g., CPU calculations)
I (Info)     - Important events (e.g., initialization complete)
W (Warning)  - Warnings (e.g., slow update cycle, high CPU)
E (Error)    - Errors (e.g., file not found, exception)

FILTERING BY LEVEL:

adb logcat -s TAG_NAME:LEVEL

Examples:
  adb logcat -s METRICS_CPU:V        (everything including verbose)
  adb logcat -s METRICS_CPU:D        (debug and above)
  adb logcat -s METRICS_CPU:I        (info and above - only important)
  adb logcat -s METRICS_CPU:E        (only errors)
